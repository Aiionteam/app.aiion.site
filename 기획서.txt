①	사용자가 메뉴바를 이용하지 않고도 메인화면의 프롬프트 창을 통하여 ai가 텍스트 전송 밑 추천, 건강상태등을 편리하게 사용하기 위해 기획
2)	개요
①	프롬프트 텍스트 입력 창
②	메뉴 바
③	AI 음성 대화하기
④	텍스트 카테고리별 파싱
3)	시스템설명
 
①	프롬프트 입력 창
A	프롬프트 입력 창을 터치하면 키패드가 호출되고 키패드를 사용하여 프롬프트를 입력할 수 있습니다.
 I	최대 5000자까지 작성할 수 있습니다.
B	프롬프트 작성 후 키패드에 리턴, 엔터, 보내기 버튼을 누르면 해당 프롬프트가 AI에게 전달됩니다.
C	전달된 프롬프트는 카카오톡 대화형식으로 메인 화면에 표시
D	라우터 AI가 텍스트를 분석하여 사용자에게 맞는 답변을 제공하거나 각 카테고리에 맞게 파싱하여 저장할 수 있습니다.
E	사용자가 프롬프트를 입력하면 MCP라우터가 그 내용을 분석합니다.
F	입력의 복잡성, 도메인, 스타일 등을 고려해 적합한 모델을 선택하거나 여러 프롬프트를 조합합니다. 결과적으로 더 정확하고 효율적인 응답을 얻을 수 있습니다.
②	텍스트 파싱
A	일기, 건강, 문화, 가계등의 관련된 텍스트를 감지하여 각 카테고리에 해당 텍스트를 전송합니다. 전송 받은 텍스트는 각각의 카테고리의 에이전트가 분석합니다. (일기, 건강, 문화,가계 시스템 설명 확인)

③	마이크 아이콘
A	아이콘을 터치하면 AI아바타 모델 화면으로 전환됩니다.
 
B	AI 아바타 대화 화면에서는 마이크 기능이 활성화됩니다.
 I	마이크가 연결 되어있지 않는 상태일 경우 비활성화
C	사용자의 음성을 인식하여 프롬프트를 텍스트로 전환시켜 작성해줍니다.
 I	음성이 인식되지 않고 3초가 지나면 자동으로 프롬프트 입력
D	프롬프트를 리퀘스트 받은 AI아바타는 사용자에게 음성으로 해당 질문 혹은 명령을 처리해줍니다.
E	마이크 아이콘을 한번 더 누르면 AI 아바타 대화 화면이 종료됩니다.
 
④	메뉴 바
 
A	사이드 바 손잡이
 I	메인 화면에서 사이드 바 손잡이를 X축 +방향으로 드래그엔 드롭 하면 숨겨져 있던 메뉴바가 메인 화면 좌측에 당겨집니다.
 II	메뉴 바에는 홈, 캘린더, 다이어리, 건강, 문화, 가계부가 존재합니다.
 III	해당 카테고리를 터치하면 터치한 카테고리 탭으로 화면이 이동합니다.
 IV	사이드 바 손잡이를 X축 -방향으로 드래그엔 드롭 하면 메뉴바가 메인 화면 좌측에서 숨겨집니다.
⑤	공통
A	닫기 버튼을 누르면 팝업 창이 종료됩니다.
B	나가기 버튼을 누르면 해당 화면의 전 화면으로 이동합니다.
C	스크롤이 존재하는 화면, 탭은 Y축 - + 터치드래그로 위아래로 이동가능합니다.

A	디폴트 캘린더는 사용자가 사용중인 기기의 해당일을 기준으로 year, month, day가 나타납니다.
B	캘린더에서 원하는 day를 선택할 수 있고 해당 day를 선택하면 하단에 일정목록의 날짜가 선택한 day를 기준으로   이 바뀝니다.
 
C	공휴일은 빨간 텍스트로 표시합니다 오른쪽은 빨간색 아이콘으로 표시
D	일정목록이나 캡션목록이 존재하는 day는 텍스트로 표시 오른쪽에 초록색 아이콘으로 표시

I	시간설정은 다이얼로 맞추어서 설정할 수 있고 하루종일을 누르면 하루종일 시간설정으로 텍스트 저장됩니다.
II	일정 저장은 최대 100개 까지 가능합니다.

⑦	MBTI
 
A	MBTI점수그래프
 I	사용자의 일기를 성격분석AI가 분석하여 MBTI를 산출해내고 각 분야는 첫번째 비교 대상 
E (외향) or I (내향), 두번째 비교 S (감각) or N (직관), 세번째 비교 T (사고) or F (감정), 
네번째 비교 J (판단) or P (인식)
 II	각 비교 대상에서 점수가 높은 축별 성향 알파벳을 조합하여 MBTI를 설정 그래프로 수치를 표시합니다. 그래프는 각 분류마다 최소0에서 100까지 존재합니다.
 III	MBTI 성격유형 정리
유형	코드	명칭	주요 특징
분석가형 (NT)	INTJ	전략가	독립적, 체계적, 장기적 계획에 강함
	INTP	사색가	논리적, 분석적, 지적 호기심 많음
	ENTJ	지휘관	리더십, 추진력, 목표 지향적
	ENTP	발명가	창의적, 토론과 도전을 즐김
외교형 (NF)	INFJ	옹호자	이상주의적, 통찰력, 깊은 공감 능력

B	문화 기록 + 취향 분석 + 맞춤 추천이 한 번에 이루어짐
3)	시스템 설명 
①	메인화면
 
A	추천 리스트
 I	여행, 영화, 공연 리스트는 최근 일기, 프롬포트를 반영하여 각 키워드를 찾아 가장 위에 배치되게 한다.
 II	최근 활동을 리포트로 정리하여 보여준다.
 III	근거가 되는 해당 일기, 프롬포트 내용을 상단에 배치하여 이용자가 알 수 있게 한다.
 IV	각 카테고리를 선택하면 더 많은 정보를 확인할 수 있다.
 V	나의 기록 버튼을 선택하여 이전 데이터들을 볼 수 있다.










②	추천 리스트 별 데이터
 
A	카테고리를 터치하면 추천 받았던 카테고리 별 기록 화면으로 이동합니다.
③	추천별 버튼
 
A	카테고리를 터치 하면 해당 추천에 대한 상세 추천 화면으로 이동합니다.
B	사용자에 맞게 순위 별로 추천 됩니다.
④	상세 추천 리스트 
 
A	선택한 상세 추천에 해당하는 상세 추천 카테고리가 보여집니다.
⑤	상세 추천에 대한 설명 
 

A	선택한 디테일의 대상에 해당하는 이미지가 보여집니다.
B	이미지는 최대10개까지 표시할 수 있습니다.
C	만약에 이미지가 존재하지 않는 디테일일 경우 “이미지가 존재하지 않습니다.”출력
D	설명 창에 손잡이 바를 위로 스와이프하면 설명에 대한 전체 내용을 확인할 수 있다.
 I	내용, 리뷰, 평점 등 확인 가능
⑥	위시 카테고리 팝업 창
 
A	위시 저장 버튼을 누르면 색이 채워지고 위시 카테고리 팝업창이 생성됩니다.
B	생성된 팝업 창에서 제목 텍스트 입력창에 디폴트 값으로 맞춤추천 키워드 네임이 자동으로 채워집니다
C	사용자가 원하면 텍스트를 임의로 정할 수 있고 글자 제한은 최대 15자까지 작성할 수 있으며 텍스트가 없으면 디폴트 값으로 입력됩니다. 그리고 새로 만들기 버튼을 누르면 새로운 상세 네임이 저장됩니다.
 
⑦	데이터 리스트
 
A	메인 화면에서 나의 기록 버튼을 선택한 후 화면입니다.
B	기록 되었던 최근 50개의 일기 기록 데이터가 보입니다.
C	상세 버튼을 터치하여 추천 받았던 기록을 열어 확인하고, 닫아 정리할 수 있습니다.
-세로 스크롤로 50개의 일기 기록을 모두 확인할 수 있습니다.
-가로 스크롤로 추천 기록들을 모두 확인할 수 있습니다.
 
⑧	위시 리스트
 
A	메인 화면 오른쪽 상단에 위시리스트 버튼을 터치하면 위시리스트 선택 화면으로 이동합니다.
B	원하는 추천 항목을 정리 관리하면 된다. 추천 항목은 여행, 연극, 영화가 디폴트로 존재합니다.
C	카테고리 버튼을 눌러 상세 위시 카테고리로 이동한다.

⑨	위시 리스트 상세 카테고리
 
A	디폴트 값으로 맞춤추천 키워드 네임이 자동으로 채워집니다
B	버튼을 2초간 눌러 사용자가 원하는 제목으로 정할 수 있고 글자 제한은 최대 15자까지 작성할 수 있으며 텍스트가 없으면 디폴트 값으로 입력됩니다. 그리고 새로 만들기 버튼을 누르면 새로운 상세 네임이 저장됩니다.

 
⑩	위시 리스트 상세 카테고리 설명
 

A	사용자가 위시로 저장해 둔 추천 목록들이 보여집니다.
B	버튼을 터치하여 각 추천에 대한 상세 설명을 보여줍니다.


 
８.	자기개발
1)	기획의도
①	현대인들은 자기 개발을 시작할 때 "무엇을 배워야 할지", "내 적성에 맞는 직무가 무엇인지" 정하지 못해 시작조차 못하는 경우가 많습니다.
②	사용자가 입력한 학습 기록과 일기를 바탕으로 학습 패턴, 지속력, 흥미도를 분석하여 자연스럽게 적합 직무를 발견하도록 돕는 것이 핵심 목적입니다.
③	"일단 시작하면 AI가 길을 찾아준다"는 철학으로 설계되었으며, 학습 → 패턴 분석 → 직무 추천 → 새로운 학습의 순환 구조를 통해 지속적인 성장을 지원합니다.
2)	개요
①	학습 기록 및 진도 관리
②	AI 기반 학습 추천 및 로드맵 생성
③	장기 학습 데이터 분석을 통한 적성 발견
④	개인 맞춤형 직무 추천 및 커리어 로드맵 제공
⑤	타 카테고리(일기, 캘린더, 가계부) 데이터와 연동한 종합 분석
3)	시스템설명
①	패스파인더 메인 화면
 
A	최근 상태 요약
 I	데이터를 기반으로 최근 학습기록과 향상된 수치, 평균 사용자 대비 학습 지속력, 적성 등의 정보를 요약해서 보여줍니다.
 II	건강 카테고리와 연동되어 컨디션 데이터를 반영합니다.
B	카테고리 이동 버튼
 I	사용자가 누른 학습, 커리어 각 카테고리로 이동합니다.
②	학습 카테고리
 
A	최근 작성된 학습 내용 표시
 I	가장 최근 기준으로 사용자가 다이어리에 학습 관련 데이터를 기반으로 작성한 내용이 표시됩니다.
B	Ai 추천 강의 버튼
 I	사용자의 학습수행능력 데이터과 알고리즘을 기반으로 사용자 수준에 맞는 현재 학습 내용과 관련된 강의를 추천해줍니다
C	새 학습 시작 이동 버튼
 I	해당 페이지로 이동합니다.
D	학습 목표 설정
 I	캘린더 탭과 연동 되어, 특정 날짜에 학습 목표를 수동으로 기입 할 수 있고, 분석된 데이터를 기반으로 자동으로 생성됩니다.
 
③	추천 학습 탭
 
A	사용자가 선택한 카테고리 맞춤 안내
 I	사용자가 선택한 카테고리 혹은 개인 학습 운동을 간략하게 안내합니다.
B	맞춤 학습 영상
 I	학습 영상이 표시됩니다.
 II	카테고리 별로 최대 5개의 영상까지 표시되며, 옆으로 밀어 화면을 이동합니다.
 
④	자기개발 탭
 
A	최근 상태 요약
 I	최근 학습을 달성한 데이터를 기반으로 추천 직무 및 강점을 표시해 줍니다.
 II	추천 직무 상세 분석보기 버튼을 누르면 해당 카테고리로 이동합니다
 III	커리어 로드맵 버튼을 누르면 해당 카테고리로 이동합니다
 IV	선택하기를 누르면 해당 선택한 직무의 정보를 <추천 직무 상세 분석보기>에서 세부내역을 볼 수 있습니다.
 
⑤	추천 직무 상세 분석탭
 
A	추천 직무와 관련한 정보들을 보여줍니다.
 
⑥	커리어 로드맵 카테고리
 
A	각각의 Phase들로 나누어, 해당 상황에 맞는 학습 영상을 추천해줍니다.
